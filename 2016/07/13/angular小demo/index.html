前言

　　最近一段时间一直在学习Angular4.x的相关知识，，这次决定用Angular4.x去模仿一个豆瓣电影页面,写下这篇博客，记录实践中的心得体会

创建项目

创建项目

ng new 项目名称 创建一个项目

ng new my-app
进入刚才创建的项目里面启动服务

cd my-app
cnpm install
ng serve –open

创建组件
ng g component my-new-component
指定目录创建 ：ng g component components/Footer
Directive
创建服务
ng g directive my-new-directive
