React第二讲课堂笔记（JSX 语法）

1. 什么是 JSX

类似 xml 的语法，用来描述组件树

JSX是可选的，并不强制要求使用。如果不用JSX，用React提供的API写的话，应该是这样的，直接调用React的API来定义组件和DOM元素。

2. 为什么要使用 JSX

你不需要为了 React 使用 JSX，可以直接使用纯粹的 JS。但我们更建议使用 JSX , 因为它能定义简洁且我们熟知的包含属性的树状结构语法。
